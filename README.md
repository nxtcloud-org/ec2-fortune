# ec2-fortune

연성대 1학기 AI/클라우드 수업 — 7주차 EC2 실습용 운세 페이지

## 구조

```
fortune-app/
├── static/
│   └── index.html      # 정적 운세 페이지 (JS로 랜덤 뽑기)
├── server.js           # 동적 운세 페이지 (Express, 방문자 카운터)
└── package.json
```

## 사용법

### 1. 패키지 설치
```bash
npm install
```

### 2. 정적 페이지 실행 (3000 포트)
```bash
npm run static
```
→ 브라우저에서 버튼 누르면 JS가 운세를 뽑아 보여줌  
→ F12 Sources에서 운세 목록이 그대로 노출됨

### 3. 동적 페이지 실행 (3000 포트)
```bash
node server.js
```
→ 버튼 누르면 `/api/fortune` 호출 → 서버가 운세 선택  
→ **누적 방문자 카운터**가 모든 사용자 공유  
→ 서버 시간 실시간 표시

### 4. 80 포트로 실행 (sudo 필요)
```bash
# server.js의 PORT를 80으로 수정
sudo node server.js
```

## 정적 vs 동적 비교 포인트

| | 정적 (static/) | 동적 (server.js) |
|---|---|---|
| 버튼 누르면 운세 바뀜 | ✅ (브라우저 JS) | ✅ (서버) |
| 공유 카운터 | ❌ | ✅ |
| 서버 시간 | ❌ | ✅ |
| F12로 운세 목록 노출 | ✅ | ❌ |

## 실습 미션

1. 운세 페이지를 80포트로 띄우기 (`sudo` + 보안그룹 80 추가)
2. 본인 이름으로 커스터마이징 (페이지 제목 + 운세 3개 이상 추가)
